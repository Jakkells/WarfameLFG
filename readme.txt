# Warframe LFG (Looking For Group)

A simple web app for organizing Warframe groups.  
Built with **Flask** and **SQLite**, this site allows Tenno to post and filter LFG requests for relic runs, Archimedea, Netracells, Eidolons, Steel Path, and more.

- SQLite backend (lightweight and easy).
- Minimal, future-proof relic search.
